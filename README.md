# 项目介绍

## 用户模块<br>

用户注册（含邮箱唯一校验）</br>
用户登录（返回 token 或 session）</br>
用户注销 </br>
用户信息获取（登录后查询自己的信息）</br>

## 商品模块（核心）

商品添加（管理员）</br>
商品列表（分页+条件搜索）</br>
商品详情（根据 id 查单个）</br>
商品上下架（可选）</br>

## 购物车模块

添加商品到购物车</br>
修改购物车商品数量</br>
删除购物车商品</br>
查询当前用户购物车</br>

## 订单模块

提交订单（从购物车生成订单）</br>
查看订单列表（用户/管理员）</br>
查看订单详情</br>
订单状态更新（如：待支付 -> 已支付）</br>
